# 课程报告大纲

## 1. 项目概述

- 项目背景：短视频推荐系统与数据结构课程设计。
- 已有数据基础：`data/bilibili_10w_pro.csv`。
- 设计目标：基于真实视频元数据模拟用户行为，并完成 F3-F7 分析。
- 技术路线：C++17 实现核心算法，少量 Python 仅作辅助分析。

## 2. 需求分析

- F1：视频信息库。
- F2：约 12000 名用户的行为模拟。
- F3：相似用户分析。
- F4：视频推荐。
- F5：热度预测。
- F6：视频聚类。
- F7：用户聚类。
- 课程限制：少调用现成库，核心算法自实现。

## 3. 总体设计

- 系统模块划分。
- 数据流设计。
- 文件目录结构。
- 命令行界面设计。
- 原始数据、模拟数据、输出结果的分层保存。

## 4. 数据设计

- 原始视频表字段说明。
- 规范化视频表。
- 模拟用户表。
- 用户行为表。
- 推荐结果表、聚类结果表、热度预测结果表。
- 核心内存结构：数组、哈希表、倒排索引、堆、稀疏向量。

## 5. 用户行为模拟方案

- 视频特征向量化。
- 视频质量分计算。
- 用户兴趣群体生成。
- 长尾活跃度分布。
- 兴趣匹配与点击生成。
- 行为漏斗：观看、完播、点赞、收藏、投币、分享。
- 时间序列注入：发布后爆发、衰减、二次爆发。
- 生成数据如何支撑 F3-F7。

## 6. 算法设计

- CSV 解析和数据构建算法。
- 倒排索引构建。
- 用户兴趣向量构建。
- 余弦相似度与 Top-K 相似用户。
- 混合推荐排序算法。
- 滑动窗口与指数平滑热度预测。
- 自实现 KMeans 视频聚类。
- 自实现 KMeans 用户聚类。

## 7. 系统实现

- 核心类设计：`Video`、`User`、`Event`。
- 各模块接口说明。
- 主要函数流程。
- 文件输入输出。
- 异常处理和边界情况。

## 8. 测试与结果分析

- 数据规模测试：视频数、用户数、行为数。
- 行为分布测试：点击、完播、点赞、收藏、投币、分享比例。
- 相似用户样例。
- 推荐结果样例。
- 热度预测曲线或表格。
- 视频聚类结果分析。
- 用户聚类结果分析。
- 性能和复杂度分析。
